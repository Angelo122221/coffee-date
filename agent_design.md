# DESIGN_AGENT.md

# Project Identity

Project Name: Coffee Date Invitation

Purpose:

Create a beautiful, personal, romantic website where a girl is invited on a coffee date through an interactive experience.

The design should feel:

* Handcrafted
* Thoughtful
* Warm
* Modern
* Playful
* Genuine

It must NOT look like:

* An AI-generated landing page
* A SaaS dashboard
* A startup website
* A corporate website
* A generic Tailwind template

The website should feel like something a college student spent time creating specifically for one person.

---

# Design Philosophy

The website should tell a story.

Instead of showing a long form immediately:

Guide the user through moments.

Each screen should feel like:

> "Someone made this for me."

The experience should be emotional before it becomes functional.

---

# Visual Inspiration

Combine elements from:

* Modern Apple product pages
* Pinterest romantic aesthetics
* Scrapbook journals
* Polaroid photo albums
* Cute coffee shop menus
* Study date aesthetics

Avoid:

* Glassmorphism overload
* Neon cyberpunk themes
* Excessive gradients
* Dark hacker aesthetics
* Generic AI-generated illustrations

# Spacing Rules

Large breathing room.

Every section should feel airy.

Never crowd content.

Use:

* Large margins
* Soft shadows
* Rounded corners

Border Radius:

24px

---

# Animations

Animations should feel soft.

Allowed:

* Fade in
* Slide up
* Floating hearts
* Gentle hover effects
* Confetti on success

Avoid:

* Aggressive bouncing
* Excessive motion
* Flashing effects

---

# GIF Usage

Use online GIFs only.

Examples:

Landing page:

* Cute coffee GIF
* Cozy cafe animation

Question screens:

* Shy anime reaction GIF
* Cute cat GIF
* Happy coffee GIF

Success page:

* Celebration GIF
* Happy heart animation

Rules:

* GIFs should support the story.
* Never place GIFs just for decoration.
* Maximum one GIF per screen.

Sources:

* Giphy
* Tenor
* LottieFiles

# Landing Page Design

Layout:

Centered content

Hero GIF at top

Large headline

Personal message

Call-to-action button

Example structure:

[ Coffee GIF ]

"Hi, I have something I'd like to ask you ❤️"

Small note

Continue Button

The first screen should create curiosity.

---

# Question Pages

Each question appears as a card.

Card Width:

Maximum 600px

Layout:

[ GIF ]

Question

Description

Options

Progress Bar

Buttons must feel large and inviting.

---

# Yes / No Screen

Yes button:

Large
Prominent
Warm accent color

No button:

Smaller

Playful movement allowed:

* Move away slightly
* Change position
* Shake gently

Never make it impossible to click.

---

# Coffee Date Selection

Display options as cards.

Example:

☕ Cozy Cafe

🚶 Coffee + Walk

📚 Study Date

🍰 Dessert & Coffee

🎁 Surprise Me

Cards should look clickable.

---

# Availability Screen

Use a beautiful calendar picker.

Do not use default browser inputs.

Preferred:

* Custom calendar component
* Warm colors
* Rounded edges

---

# Photo Upload

Design like a Polaroid frame.

Before Upload:

📸 Upload a photo

After Upload:

Show image preview

White border

Polaroid style

---

# Success Page

Must feel rewarding.

Layout:

Confetti

Celebration GIF

Thank You Message

Summary Card

Cute closing note

Example:

"Thank you for saying yes ❤️"

"You just made my day."

---

# Accessibility

Maintain:

* Proper contrast
* Keyboard navigation
* Mobile responsiveness
* Readable text

Do not sacrifice usability for aesthetics.

---

# Mobile First

Primary target:

Mobile devices

Breakpoints:

Mobile
Tablet
Desktop

The website should feel perfect on phones.

---

# AI Detection Avoidance

Do NOT generate:

* Generic marketing copy
* Startup buzzwords
* Corporate language
* Perfectly symmetrical layouts everywhere

Instead:

Use:

* Small imperfections
* Personal notes
* Human-written messages
* Storytelling

The site should feel like it was created by a real person asking someone on a date.
