# Coffee Date Invitation

An Astro-based romantic invitation site for collecting coffee date responses.

## Run locally

1. Install dependencies with `npm install`
2. Copy `.env.example` to `.env`
3. Fill in your Supabase URL and anon key
4. Start the app with `npm run dev`

## What’s included

- Landing page
- Multi-step invite flow
- Photo upload form path
- Supabase-ready data helpers
- Thank-you page
- Admin dashboard scaffold
